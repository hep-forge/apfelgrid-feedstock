Feedstock for [apfelgrid](http://nhartland.github.io/APFELgrid/) — part of [hep-forge](https://anaconda.org/hep-forge).

Ultra-fast theory predictions for collider observables

## Install

```bash
conda install -c hep-forge -c conda-forge apfelgrid
```
