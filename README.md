Feedstock for [apfelgrid](http://nhartland.github.io/APFELgrid/) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

Ultra-fast theory predictions for collider observables

## Architectures

| Architecture | Latest published |
|--------------|------------------|
| linux-amd64 (`linux-64`) | ✅ `1.0.1` |
| linux-arm64 (`linux-aarch64`) | ❌ not published |

_As of the last feedstock render; the badges above are live._


## Install

```bash
conda install -c hep-forge -c conda-forge apfelgrid
```
